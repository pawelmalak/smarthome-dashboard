## Smart Home Dashboard

Mockup application made as part of recruitment process

### Technologies

- React
- Node.js
- Express
- Typescript

### How to run

```bash
git clone http://github.com/pawelmalak/smarthome-dashboard
cd smarthome-dashboard
npm run init
npm run dev

# visit localhost:3000
```
